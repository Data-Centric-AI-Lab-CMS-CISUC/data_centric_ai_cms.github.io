## 🛠️ Software & Tools

We develop and maintain several open-source tools and frameworks to support data-centric AI research.

| Tool | Description | Link |
| :--- | :--- | :--- |
| **Fairforge Prototype** | Unified Framework for Responsible AI (AutoML tool). | [Access Tool](https://fairforge.streamlit.app/) |
| **mdatagen** | Artificial generation of missing data (MNAR/MAR/MCAR). | [PyPI](https://pypi.org/project/mdatagen/) |
| **PMIVAE** | Partial multiple imputation using Variational Autoencoders. | [GitHub](https://github.com/ricardodcpereira/PMIVAE) |
| **VAE-BRIDGE** | VAE filter for Bayesian Ridge Imputation. | [GitHub](https://github.com/ricardodcpereira/VAE-BRIDGE) |
| **SAEI** | Siamese autoencoder architecture for imputation. | [GitHub](https://github.com/ricardodcpereira/SAEI) |
| **FairML** | A fairness-aware imputation algorithm. | [GitHub](https://github.com/ArthurMangussi/FairML) |
| **FAIRFATE** | Fair Federated Learning aggregation method. | [GitHub](https://github.com/teresalazar13/FAIRFATE) |
