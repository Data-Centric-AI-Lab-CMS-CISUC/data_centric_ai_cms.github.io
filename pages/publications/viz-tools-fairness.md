## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/file-text.svg" width="20" height="20" style="vertical-align: middle;"> Abstract
Designing visualization tools for group fairness analysis requires careful consideration of transparency and usability. This paper provides a set of guidelines for building effective visualization interfaces that help researchers identify and mitigate bias in binary classification models.

## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/quote.svg" width="20" height="20" style="vertical-align: middle;"> Full Citation
> António Cruz, Teresa Salazar, Manuel Carvalho, Catarina Maçãs, Penousal Machado, Pedro Henriques Abreu, **"Guidelines for designing visualization tools for group fairness analysis in binary classification"**, *Artificial Intelligence Review*, 58 (182), 38 pages, 2025.
